# 📊 Customer Churn Prediction & Sales Dashboard

A complete **Data Analytics and Machine Learning project** that predicts customer churn, analyzes sales trends, and visualizes insights using an interactive **Streamlit dashboard**.

---

## 🧠 Project Overview

The **Customer Churn Prediction & Sales Dashboard** helps businesses understand customer behavior, identify churn risks, and improve retention strategies.  
It combines **Machine Learning**, **Time-Series Forecasting**, and **Data Visualization** in one unified dashboard.

---

## 🚀 Features

### 🔹 1. Data Acquisition & Preprocessing
- Collected data from CRM systems and transaction logs.
- Handled missing values and normalized numeric features.
- Encoded categorical columns using one-hot encoding.
- Performed feature engineering — purchase frequency, average spend, and customer lifetime.

### 🔹 2. Churn Prediction Models
- **Supervised Models:** Logistic Regression, Random Forest, XGBoost, LightGBM.  
- **Deep Learning Model:** Artificial Neural Network (ANN) built using TensorFlow/Keras.  
- Evaluated models with **accuracy**, **F1-score**, and **ROC-AUC** metrics.

### 🔹 3. Sales Trend Analysis
- Visualized **monthly and yearly sales trends**.  
- Identified **top-performing products**.  
- Analyzed **revenue impact** caused by customer churn.

### 🔹 4. Forecasting (Prophet)
- Used **Facebook Prophet** to predict sales for the next 12 months.  
- Displays trend and seasonality components interactively.

### 🔹 5. Customer Segmentation
- Used **K-Means clustering** to group customers based on behavior.  
- Applied **DBSCAN** for anomaly detection and outlier analysis.

### 🔹 6. Streamlit Dashboard
- Real-time **customer churn prediction** by Customer ID.  
- **Sales trend visualization** (line and bar charts).  
- **Revenue distribution** between retained and churned customers.  
- **Forecasting results** shown interactively.
